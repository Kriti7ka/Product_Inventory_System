# Product Inventory System

## Overview
Product Inventory System is a Java-based application that helps manage product details and inventory efficiently. The system allows users to add, update, delete, and view products while maintaining inventory records.

## Features
- Add new products
- Update product information
- Delete products
- View product details
- Search products
- Inventory management
- User-friendly console interface

## Technologies Used
- Java
- Eclipse IDE
- Git & GitHub

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/Kriti7ka/Product_Inventory_System.git
```

2. Open the project in Eclipse.

3. Run the main Java file.
